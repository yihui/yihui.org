简单罗列一下这两个月发生的事情。

        ```bash
        #!/bin/sh
    
        while true
        do
          ifconfig | grep -q 10.2.13 && true || (sudo rmmod iwlwifi;
             sleep 1; sudo modprobe iwlwifi; sleep 30;
             echo rebooted wireless)
          sleep 1
        done
        ```
